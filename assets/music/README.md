# 背景音乐说明

## 如何添加音乐

请下载一首新年庆祝音乐的MP3文件，重命名为 `new-year-2026.mp3`，然后放在这个目录下。

## 推荐的音乐来源

### 1. Pixabay Music（推荐）⭐
- 网址：https://pixabay.com/music/search/new%20year/
- 免费商用，无需署名
- 搜索关键词："new year", "celebration", "fireworks"
- 推荐风格：Electronic 或 Cinematic

### 2. Chosic
- 网址：https://www.chosic.com/free-music/celebration/
- 免费商用音乐，CC0许可
- 分类：Celebration / New Year

### 3. Mixkit
- 网址：https://mixkit.co/free-stock-music/tag/celebration/
- 免费庆祝音乐库

## 音乐要求

- 文件名：`new-year-2026.mp3`（必须是这个名字）
- 文件格式：MP3
- 比特率：128kbps
- 文件大小：< 5MB
- 风格：纯音乐（无歌词），节日庆典风格
- 时长：2-4分钟（会循环播放）

## 下载步骤

1. 访问上述任一网站
2. 搜索 "new year celebration" 或 "fireworks"
3. 选择一首纯音乐曲目
4. 点击下载按钮
5. 将下载的文件重命名为 `new-year-2026.mp3`
6. 放入 `/Code/Wishes/HappyNewYear2026/assets/music/` 目录

## 注意事项

- ⚠️ 音乐文件不会推送到Git（已在.gitignore中忽略）
- ⚠️ 请确保音乐文件无版权问题
- ⚠️ 如果没有音乐文件，页面仍然可以正常显示，只是没有背景音乐
- ✅ 音乐会在倒计时结束后自动播放（如果浏览器允许）
- ✅ 如果自动播放失败，会显示"播放音乐"按钮

## 测试

添加音乐文件后，刷新页面，等待倒计时结束（或手动修改 `countdown.js` 中的 `TARGET_DATE` 为当前时间之后几秒），即可看到完整的庆祝效果！
